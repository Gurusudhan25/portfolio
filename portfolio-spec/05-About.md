# About Section Specification

## Objective

The About section should answer one important question:

"Who is Gurusudhan beyond the technologies listed on the resume?"

This section should explain how I think as an engineer, how I solve problems, and what motivates me to build software.

Avoid generic self-introductions.

Avoid cliché statements like:

- Passionate Developer
- Coding since childhood
- Love solving problems
- Hardworking and dedicated

Instead, tell an authentic engineering story.

# AI Instructions

Do not generate generic biography content.

Generate a narrative that feels personal, professional, and engineering-focused.

Focus on:

How I think

How I solve problems

How I collaborate

How I improve software

How I continue learning

The About section should make recruiters feel confident that I can contribute effectively to modern engineering teams.
