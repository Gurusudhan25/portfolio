# Footer & Ending Experience Specification

## Objective

The footer should not feel like the end of a webpage.

It should feel like the ending scene of a premium product launch.

This section should leave a lasting impression on recruiters and hiring managers.

Visitors should pause for a moment before leaving the website.

The footer should become one of the most memorable parts of the portfolio.

---

# Overall Experience

As the visitor reaches the final section:

The entire atmosphere should change.

Scrolling should slow naturally.

The background becomes more immersive.

Animations become calmer.

The typography becomes larger.

The ending should feel intentional.

---

# Layout

The footer occupies the full viewport width.

Minimum height

100vh

Preferred

120vh

The footer should not look compressed.

Give it generous breathing space.

---

# Background

The background should be generated from my uploaded profile image.

Image Source

Use the profile image provided in the project assets.

Never place the original portrait directly as a background.

Instead create an artistic composition.

Possible processing

Background removal

Monochrome conversion

Gradient color grading

Soft blur

Noise overlay

Grain texture

Large radial gradients

Light bloom

Low-opacity silhouette

Color blending

Soft vignette

Glass overlays

The portrait should feel embedded into the design.

It should appear almost like artwork.

---

# Background Motion

Very subtle animation.

Possible ideas

Animated grain

Slow gradient movement

Moving light

Soft blur transition

Very slow parallax

Nothing should distract from the typography.

The portrait should merge naturally into the typography.

Portrait behind typography.

Portrait blended into gradients.

Typography remains the hero.

# Scroll Ending

When the user reaches the bottom

Animations slow.

The background stabilizes.

The name becomes fully visible.

Everything feels calm.

No abrupt ending.

---

# Mobile Experience

The footer should be redesigned for mobile.

Typography scales properly.

Portrait remains visible.

Spacing stays generous.

Buttons remain touch friendly.

The experience should still feel premium.

---

# Accessibility

Readable typography

Proper contrast

Semantic footer

Keyboard accessible

Screen reader friendly

Reduced motion support

---

# Performance

Optimize portrait image.

Generate responsive image variants.

Lazy-load non-critical assets.

Use GPU accelerated animations.

Maintain high Lighthouse performance.

---

# AI Instructions

The footer should become the emotional ending of the portfolio.

Do not build a standard footer.

Avoid simple copyright sections.

Avoid generic layouts.

The visitor should leave remembering:

the typography,

the atmosphere,

the motion,

and the craftsmanship.

The footer should feel like the final scene of a beautifully designed product experience.

It should make recruiters smile before closing the tab.
