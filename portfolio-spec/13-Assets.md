# Asset Management Specification

## Objective

This document defines every asset used throughout the portfolio.

The AI should never use placeholder assets if official assets are available.

Always prefer official resources, high-quality images, SVG icons, and optimized formats.

Every asset should be responsive, optimized, and lazy-loaded where appropriate.

# AI Instructions

Never use random stock images.

Never use placeholder avatars.

Use only assets that support the engineering story.

Every visual should reinforce quality, professionalism, and attention to detail.

The portfolio should feel like a carefully designed product rather than a collection of unrelated assets.
